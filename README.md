## âš™ï¸ **CUSTOMIZATION & CONFIGURATION**

### ğŸ¨ **Enhanced Title Screen Configuration**

Customize the 80% improved title screen through:

#### **1. JSON Configuration File**
Edit `assets/data/enhancedTitleConfig.json`:

```json
{
    "performanceMode": false,    // Enable for lower-end devices
    "particleCount": 50,         // Number of background particles
    "trailEffects": true,        // Enable logo motion trails
    "lightingEffects": true,     // Enable glow and ray effects
    "waveEffects": true,         // Enable wave distortions
    "dynamicColors": {
        "primary": [255, 51, 255, 255],    // Magenta theme
        "secondary": [51, 51, 204, 255],   // Blue accent
        "accent1": [255, 51, 255, 255],    // Magenta highlight
        "accent2": [51, 255, 51, 255]      // Green accent
    }
}
```

#### **2. Project.xml Defines**
Add compilation flags in `Project.xml`:

```xml
<!-- Enhanced Title Screen -->
<haxedef name="ENHANCED_TITLE" />

<!-- Performance optimization -->
<!-- <haxedef name="ENHANCED_TITLE_PERFORMANCE" /> -->

<!-- Original features -->
<haxedef name="LUA_ALLOWED" />
<haxedef name="VIDEOS_ALLOWED" />
<haxedef name="MODS_ALLOWED" />
```

## Cool new Chart Editor changes and countless bug fixes
![](https://github.com/ShadowMario/FNF-PsychEngine/blob/main/docs/img/chart.png?raw=true)
* You can now chart "Event" notes, which are bookmarks that trigger specific actions that usually were hardcoded on the vanilla version of the game.
* Your song's BPM can now have decimal values
* You can manually adjust a Note's strum time if you're really going for milisecond precision
* You can change a note's type on the Editor, it comes with two example types:
  * Alt Animation: Forces an alt animation to play, useful for songs like Ugh/Stress
  * Hey: Forces a "Hey" animation instead of the base Sing animation, if Boyfriend hits this note, Girlfriend will do a "Hey!" too.

## Multiple editors to assist you in making your own Mod

* Working both for Source code modding and Downloaded builds!

## Story mode menu rework:
![](https://i.imgur.com/UB2EKpV.png)
* Added a different BG to every song (less Tutorial)
* All menu characters are now in individual spritesheets, makes modding it easier.

## Credits menu
![Screenshot_1](https://user-images.githubusercontent.com/44785097/144632635-f263fb22-b879-4d6b-96d6-865e9562b907.png)
* You can add a head icon, name, description and a Redirect link for when the player presses Enter while the item is currently selected.

## Awards/Achievements
* The engine comes with 16 example achievements that you can mess with and learn how it works (Check Achievements.hx and search for "checkForAchievement" on PlayState.hx)

## Options menu:
* You can change Note colors, Delay and Combo Offset, Controls and Preferences there.
 * On Preferences you can toggle Downscroll, Middlescroll, Anti-Aliasing, Framerate, Low Quality, Note Splashes, Flashing Lights, etc.

## Other gameplay features:
* When the enemy hits a note, their strum note also glows.
* Lag doesn't impact the camera movement and player icon scaling anymore.
* Some stuff based on Week 7's changes has been put in (Background colors on Freeplay, Note splashes)
* You can reset your Score on Freeplay/Story Mode by pressing Reset button.
* You can listen to a song or adjust Scroll Speed/Damage taken/etc. on Freeplay by pressing Space.
